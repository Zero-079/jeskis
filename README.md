# Jeskis

Un metroidvania oscuro y surreal ambientado en una ciudad industrial decadente.

## Desarrollo

```bash
# Instalar dependencias
npm install

# Servidor de desarrollo
npm run dev

# Construir para producción
npm run build

# Vista previa de producción
npm run preview
```

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
